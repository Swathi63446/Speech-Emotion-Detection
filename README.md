# 🎙️ Speech Emotion Recognition (SER)

This project uses Machine Learning to detect emotions from human speech. By analyzing the frequency and time-based features of an audio clip, the system can predict the speaker's emotional state.

## 🚀 Features
- **Audio Analysis:** Extracts 40 MFCC features from voice recordings.
- **Deep Learning Model:** Uses a trained LSTM (Long Short-Term Memory) network for high-accuracy sequence classification.
- **Web Interface:** A simple, interactive UI built with Streamlit for easy testing.
- **Supported Emotions:** Neutral, Calm, Happy, Sad, Angry, Fearful, Disgust, and Surprised.
